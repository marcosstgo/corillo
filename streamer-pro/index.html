Características

Hecho para correr sin supervisión

No es un reemplazo de OBS. Es una herramienta diferente — para transmitir contenido pregrabado de forma continua y automática.

Aceleración por GPU

Detecta automáticamente NVIDIA NVENC, Intel QSV o AMD AMF. Si no hay GPU compatible, usa libx264 como fallback sin configuración manual.

Perfiles de calidad

480p Bajo · 720p Medio · 1080p Alto · 1080p60 Fluido · Personalizado. Controla bitrate de video/audio, preset, resolución y FPS.

Reconexión automática

Si la conexión se cae, reconecta solo — 2s → 4s → 8s → 16s → 30s. Loop infinito para que el canal siga vivo 24/7 sin que tengas que estar pendiente.

3 modos de entrada

Una carpeta de videos se convierte en playlist automática con loop y shuffle. O transmite un solo archivo o una URL de stream. MP4, MKV, MOV, AVI, WEBM.

Métricas en vivo

Bitrate, FPS, CPU, memoria, GPU y velocidad de procesamiento en tiempo real. Historial de eventos con buffer circular para rendimiento óptimo.

Stream key segura

Cifrado con DPAPI (ProtectedData) en Windows. Fallback AES para compatibilidad. Configuración guardada automáticamente. Sin telemetría.

Compatibilidad

Transmite a cualquier plataforma

Ingresa la URL RTMP y tu stream key. Streamer Pro valida la conexión automáticamente antes de iniciar.
