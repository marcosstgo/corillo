Key Features

EVERYTHING YOU NEED TO STREAM

Streamer Pro bundles all the tools a broadcaster needs — no extra installs, no configuration headaches. Just open, configure, and go live.

Multi-Platform RTMP Streaming

Stream simultaneously or selectively to Corillo, Twitch, Kick, YouTube, or any custom RTMP endpoint. One interface, multiple destinations.

Built-in FFmpeg Engine

FFmpeg and FFprobe are bundled directly with the app. No external installs, no PATH configuration — everything works out of the box.

Online Sources + Local Folder Playlist

Add online stream URLs or point to a local folder of videos. Streamer Pro builds and streams a concat playlist seamlessly.

Random + Loop Playlist Mode

Randomize your playlist order or loop continuously — perfect for automated 24/7 channels or looping highlight reels.

Quality Profiles

Preset profiles for 720p and 1080p, plus full control over bitrate and FPS. Match any platform's recommended settings in seconds.

Aspect-Ratio Safe Scaling

Intelligent padding and scaling ensures your video is never stretched or distorted, regardless of the source resolution.

FFmpeg Process Management

Smart detection and lifecycle management for the FFmpeg process — no orphan processes left behind when you stop or restart a stream.

Technical Overview

HOW IT WORKS

Streamer Pro is a native Windows desktop application engineered for reliability and performance. Built on modern .NET technologies with an async-first architecture.

 Stack & Architecture

  • Windows desktop app built with C# / .NET (WPF) in Visual Studio 2022.
  • Uses FFmpeg to encode and stream video and audio to RTMP endpoints via the standard RTMP protocol.
  • Supports folder-based playlists by dynamically generating FFmpeg concat playlist files.
  • Ships with ffmpeg.exe and ffprobe.exe bundled next to the executable — no external install needed.
  • Source list and settings stored in a user-editable JSON file under %AppData% for per-user portability.
  • UI remains fully responsive during streaming using async process handling and cancellation tokens — no freezes, no blocking calls.

Use Cases

WHO IS IT FOR?

Streamer Pro is designed for anyone who needs reliable, automated, or multi-platform streaming without the complexity of a full production suite.

Content Creators
Stream pre-recorded content — movies, highlights, clips — to multiple platforms at once with zero manual effort.
Churches & Communities
Broadcast services, events, or community content reliably without expensive broadcast gear.
Event Broadcasters
Set up event streams quickly, with platform selection and stream key management in one place.
Automated 24/7 Channels
Run looping playlists around the clock — music channels, news tickers, highlight reels — fully automated.
Digital Signage
Loop waiting room videos, promotional content, or branded playlists through a simple, reliable stream.
Small Teams
For teams that need "set it and run" streaming — minimal setup, maximum reliability, no technical expertise required.
