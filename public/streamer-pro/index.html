Key Features

FEATURES

All tools included — no extra installs, no PATH configuration. Open, configure, stream.

Multi-Platform RTMP Streaming

Stream simultaneously or selectively to Corillo, Twitch, Kick, YouTube, or any custom RTMP endpoint. One interface, multiple destinations.

Integrated FFmpeg Downloader

No FFmpeg on your system? No problem. The app detects it's missing and downloads it automatically (~35 MB) with a progress indicator. No PATH configuration needed.

Online Sources

Relay or rebroadcast any stream URL — YouTube, Twitch, or any HTTP/RTMP source. Pull it in and push it to your destination with full quality control.

Scheduled Start

Plan a stream for a local date and time while Streamer Pro stays open or minimized to tray. Great for playlist channels, event rollouts, and unattended starts.

Stable Folder Streaming

Folder mode uses a Persistent Streamer Pipeline — a long-lived RTMP streamer stays connected while per-clip suppliers feed content into it, improving reliability for looping and randomized local playlists.

Single Local File

Stream a single video file directly — no playlist setup needed. Load the file and go live instantly. Ideal for events, presentations, or one-off broadcasts.

Random + Loop Playlist Mode

Randomize your playlist order or loop continuously — perfect for automated 24/7 channels or looping highlight reels without any manual intervention.

Highlight Mode

Built for SteelSeries Moments, NVIDIA Highlights, and AMD ReLive. Watches clip folders in real time — each new clip hits your stream within minutes of the play, and the last clip loops so your channel never goes black.

Quality Profiles

Five ready-to-use profiles: 480p, 720p, 1080p, 1080p60 Fluido, and Custom. Full control over bitrate, FPS, and resolution in one click.

Hardware Acceleration

Auto-detects NVIDIA (NVENC), AMD (AMF), and Intel (QSV) GPUs. Enable hardware encoding to drastically reduce CPU load during high-bitrate or 1080p60 streams.

Real-time Performance Monitoring

Live stats bar shows bitrate, FPS, CPU, GPU, memory, and encoding speed while streaming — spot issues before your viewers do.

Aspect-Ratio Safe Scaling

Intelligent padding and scaling ensures your video is never stretched or distorted, regardless of the source resolution.

Favorites & System Tray

Save your go-to quality settings as favorites for instant recall. Minimize to system tray and keep streaming in the background — the stream never stops.

Logo Overlay

Add a custom logo or watermark to your stream. Choose the corner position and size — FFmpeg burns it in automatically on every frame, in every mode.

Screen Capture Mode

Stream directly from your desktop or any monitor — no OBS required. Uses Windows Desktop Duplication with GPU hardware encoding (NVIDIA, AMD, or Intel) for a lightweight, zero-overhead capture pipeline.

Auto-Update

Streamer Pro checks for new versions on every launch. When an update is available, a badge appears in the header — one click downloads and installs it automatically, no manual downloads needed.

EN / ES Language Toggle

Switch the entire interface between English and Spanish with one click. The preference is saved automatically — every label, button, and message updates instantly across all windows.

Philosophy

WHY IT FEELS DIFFERENT

Built for operational streaming, not scene production. If OBS is the production suite, Streamer Pro is the focused RTMP control room.

  • Run clip channels and loop content reliably without scene management overhead.
  • Stream new highlights automatically the moment they're saved to disk.
  • Start streams on schedule — leave it minimized, it handles the rest.
  • One clean dashboard for RTMP delivery — no scene trees, no source nesting.
  • Direct RTMP publishing without OBS complexity or a browser source in sight.
Technical Overview

HOW IT WORKS

Streamer Pro is a native Windows desktop application engineered for reliability and performance. Built on modern .NET technologies with an async-first architecture.

 Stack & Architecture

  • Windows desktop app built with C# / .NET (WPF) in Visual Studio 2022.
  • Uses FFmpeg to encode and stream video and audio to RTMP endpoints via the standard RTMP protocol.
  • Folder mode uses a Persistent Streamer Pipeline for improved long-running stability — a long-lived RTMP streamer stays connected while per-clip suppliers feed content into it. Highlight Mode extends this with a dual-process MPEG-TS pipe for real-time clip injection.
  • Detects missing ffmpeg.exe / ffprobe.exe on startup and downloads them automatically (~35 MB) — no manual install or PATH setup needed.
  • Source list and settings stored in a user-editable JSON file under %AppData% for per-user portability.
  • UI remains fully responsive during streaming using async process handling and cancellation tokens — no freezes, no blocking calls.
  • Frame-based watchdog monitors real FFmpeg output frames. If no new frames are produced for 20 seconds, the process is restarted automatically — detects true freezes without false positives.
  • Auto-recovery in all modes — File, Folder, Highlight, and Capture modes reconnect automatically after any unexpected drop, keeping your channel live with minimal downtime.
