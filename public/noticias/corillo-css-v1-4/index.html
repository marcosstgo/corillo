CORILLO CSS ENTRA EN V1.4

El framework del proyecto sigue creciendo con componentes que ya no viven solo en demos, sino dentro del producto real.

Qué cambió

Corillo CSS ya no está en una etapa de experimento. La versión 1.4 sigue empujando el framework hacia componentes que salen del site real, especialmente del player.

  • Entraron piezas nuevas como crl-player-shell, crl-chat-panel, crl-player-dropdown, crl-player-rail y crl-player-vods.
  • La documentación se puso al día con demos reales y el conteo actual del framework.
  • El objetivo sigue siendo claro: menos CSS suelto por página y más sistema compartido.

Por qué importa

Cuando el framework nace desde necesidades reales del producto, el código se vuelve más fácil de mantener y las páginas se sienten más consistentes. Ese es el tipo de base que queremos para Corillo.
