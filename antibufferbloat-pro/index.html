Características

Todo lo que necesitas

Anti-BufferBloat Pro ajusta los parámetros de red de Windows que Windows no expone — para que tengas la menor latencia posible mientras juegas o streameas.

🎮
Perfil Gaming
Un click aplica el perfil de latencia mínima: Nagle OFF, Auto-Tuning restricted, timestamps OFF, InitialRTO 1s. Reset restaura los valores por defecto de Windows.
⚡
Acciones TCP
Control granular: Auto-Tuning Normal / Restricted / OFF, RSS ON/OFF, Backup y Restore de configuración TCP, Diagnóstico completo.
🖥
PC Gaming Boost
Ultimate Performance, deshabilitar Xbox DVR, activar GPU Hardware Scheduling (HAGS) y añadir exclusión a Windows Defender.
📈
Monitor en tiempo real
Gráfica de ping con los últimos 120 puntos. Ping, jitter, packet loss, auto-tuning state, IP externa e IP interna actualizados cada 5 segundos.
📡
Test de BufferBloat
3 fases bajo carga real (Idle · Download · Upload, ~60s). 6 streams paralelos — satura conexiones gigabit. Selección automática del servidor más cercano. Calificación A–F.
↑
Auto-update
Detecta nuevas versiones en GitHub al iniciar. Descarga e instala con un click — sin desinstalar ni buscar manualmente.
